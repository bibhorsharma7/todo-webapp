## To Do List Web-App

To run the web application locally:

```bash
npm install
# then
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the website locally.

## Tech Stack Used
- Next.js ^14
- React ^18
- tailwindcss
- typescript

## Files
